# RegulIoT

**RegulIoT** — автономное устройство для мониторинга и анализа безопасности инфраструктуры локального Интернета вещей (IoT), применяемой в системах «умного дома», на производственных объектах и в других распределённых средах.

Устройство предназначено для обнаружения подключённых IoT-устройств, анализа их потенциальных уязвимостей на основе базы CVE и информирования пользователя о выявленных рисках безопасности.

---

## Назначение изделия

RegulIoT обеспечивает:

- централизованный контроль IoT-инфраструктуры в локальной сети;
- регулярную проверку устройств на наличие известных уязвимостей;
- интеллектуальную оценку уровня риска эксплуатации уязвимостей;
- уведомление пользователя о потенциальных угрозах безопасности.

Устройство функционирует автономно и не требует постоянного подключения к внешним сервисам, за исключением периодического обновления базы уязвимостей.

## Принцип работы

1. **Установка и первичное подключение**

   Установите устройство RegulIoT в локальном пункте управления с наиболее стабильным сетевым подключением.

   При первом запуске подключите устройство к сети Интернет, в которой функционируют устройства Интернета вещей.

2. **Сканирование локальной сети**

   В разделе «Все устройства» выполните сканирование локальной сети для обнаружения подключённых IoT-устройств и формирования перечня контролируемых узлов.

3. **Проверка базы уязвимостей**

   В разделе «База уязвимостей» осуществляется работа с локальной базой CVE, содержащей сведения об известных уязвимостях за последние два года. База данных обновляется автоматически при наличии сетевого подключения.

4. **Мониторинг уведомлений**

   Устройство формирует уведомления при обнаружении уязвимостей с повышенным уровнем риска. Регулярный мониторинг уведомлений позволяет своевременно принять меры и предотвратить эксплуатацию уязвимостей в сети.

## Срок службы

Нормативный срок службы устройства составляет **4 года** при соблюдении условий эксплуатации, указанных в руководстве пользователя.
